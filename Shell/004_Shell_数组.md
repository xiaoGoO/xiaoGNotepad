# 数组
- 序
    > - 只支持一维数组
    > - 下标由0开始的数组
- 数组的定义
    - 一般用**括号**“()” 来表示数组，每个元素用**空格**隔开 
        ```code shell
        # 变量名称=(元素0 元素1 元素2 ... 元素n-1)
        # 定义数组变量的方法有三种
        # 类型一
        array=("小关1号" "小关2号" "小关3号")
        #类型二
        array2=(
            "小燕1号"
            "小燕2号"
            "小燕3号"
        )
        #类型三
        array3[0]="xiaoG一号"
        array3[1]="xiaoG二号"
        array3[2]="xiaoG三号"
        ```
- 读取数组
    - 读取数组的格式为 ${数组[下标]}
        ```code shell
        array=("小关1号" "小关2号" "小关3号")
        # 读取数组第一个值
        echo ${array[0]}
        # 读取数组所有元素
        echo ${array[@]}

        # 读取数组长度
        echo ${#array[@]}
        #或者
        echo ${#array[*]}

        #获取数组第n个元素的长度 
        echo ${#array[n]}
        ```
